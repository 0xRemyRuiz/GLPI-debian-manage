GLPI install and safe update debian scripts
===========================================

NOTE: GLPI files will be installed in /home/www-data so the files can be set on a separated partition

 - To install run `sudo ./install_glpi`
 - To update run `sudo ./update_glpi`
 - To list live and current GLPI version run `sudo ./list_glpi_versions`
 - To make a simple backup `./backup_scripts`
